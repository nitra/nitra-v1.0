Как я работаю?

Шаг 1  Диагностика.

При сканировании сайта я определяю неэффективные зоны в безопасности и производительности на Вашем сайте.

 

Шаг 2   Прототип.

После сканирования сайта, подбирается уникальная комбинация улучшений для повышения безопасности и увеличения скорости сайта, которая тестируется на специально созданном прототипе Вашего сайта. Для этого используются сверхмощные серверные решения Google Подробнее, что дает возможность в режиме реального времени проверить качество внедрения HTTPS, HTTP/2, OCSP, да и любой другой технологии. Такой уровень проверки не доступен специалисту. Никаких тестов на "кроликах" - это же твой сайт!

 

Шаг 3   Апгрейд сайта.

Сайт - не арт-объект. Сайт должен эффективно работать, а для этого он должен быть усилен технологией, которая поможет его сделать производительнее и безопаснее.

  1. Определяю индивидуальную для твоего сайта комбинацию HTTPS, HTTP/2, IPV6, ALPN, ECDSA, PFS, HSTS, OCSP и других алгоритмов. При их настройке я делаю тысячи тестов, чтобы проверить работоспособность сайта, каждой страницы, каждого блока.

  2. Исправляю несовершенства, которые мешают поисковикам "считывать" твой сайт. Каждый месяц буду отправлять отчет с историей исправлений и усовершенствований.
